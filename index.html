Built for the first wave of controller-first PC games.

OkamaOS boots into a focused game shell, expects a controller, and treats every .ok package like a console cartridge. Okama Studio lets anyone build and export games in the browser with AI assistance. The catalog is still small — that is the opportunity for early creators and early players alike.

Update channel

Keep every box on the latest build.

The public JSON feed is what OkamaOS devices check for OS and game updates when UPDATE_NOTIFICATIONS=yes. Download a .okupdate bundle directly from here or run okama-update check from the dev console. Rollback is always one command away.
