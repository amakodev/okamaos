Get OkamaOS

Download, install, and play — three ways in.

Recommended Option A — Live USB

Boot from USB stick

Flash OkamaOS to any USB drive and boot your PC directly into the game console. No installation needed — just plug in and play.

  1. Download the OkamaOS image from GitHub Releases.
  2. Download Balena Etcher (free, balena.io/etcher) — works on Windows, Mac, and Linux.
  3. Insert a USB drive (8 GB+), open Etcher, select the OkamaOS image and your USB drive, then click Flash.
  4. Restart your PC. Press F12 (or F2 / Del) at the BIOS screen and choose your USB drive from the boot menu.
  5. Plug in a gamepad and enjoy! 🎮

Try it safely Option B — VirtualBox

Run inside VirtualBox

Test OkamaOS without touching your PC. VirtualBox is free and runs on Windows, Mac, and Linux. Great for trying it out first.

  1. Download VirtualBox (free, virtualbox.org) and install it.
  2. Download the OkamaOS image from GitHub Releases.
  3. Open VirtualBox → New. Name it “OkamaOS”, Type: Linux, Version: Other Linux (64-bit).
  4. Set RAM to 2 GB (2048 MB) and create a 20 GB virtual hard disk.
  5. In Settings → Storage, attach the OkamaOS image as an optical drive. Start and boot!
